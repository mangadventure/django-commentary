=================
Django Commentary
=================

.. module:: commentary
   :synopsis: A fork of "django-contrib-comments".

.. highlightlang:: html+django

This framework is a fork of ``django-contrib-comments`` which used to be
included in Django. It can be used to attach comments to any model, so
you can use it on blog entries, photos, book chapters, or anything else.
